CRUD Feito Utilizando Node.js + MySQL
# O Que é CRUD?
CRUD é a sigla para:
<ul>
  <li>C -> CREATE</li>
  <li>R -> READ</li>
  <li>U -> UPDATE</li>
  <li>D -> DELETE</li>
</ul>
São as operações básicas dos bancos de dados onde os verbos realizam a ação de criar, ler, atualizar e excluir dados respectivamente.
